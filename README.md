# AI Mind Games Arena (1v1 Shooter)

Welcome to the Arena. This is a tactical 1v1 shooter where you face an AI that learns from your habits.

## 🎮 How to Play

1.  **Choose Loadout**: Select your armor (affects HP and speed) and primary weapon.
2.  **Fight**: Move with **WASD** and aim/shoot with the **Mouse**.
3.  **Upgrade**: If you defeat the AI, you earn an upgrade.
4.  **Repeat**: The AI levels up every round, becoming smarter, faster, and more predictive.
5.  **Death is Final**: If you die, your score is saved to the leaderboard and you start over.

## ⌨️ Controls

*   **W/A/S/D**: Move
*   **Mouse Move**: Aim
*   **Left Click**: Shoot
*   **R**: Manual Reload (Automatic when empty)

## 🧠 AI Learning System

The AI tracks your movement tendencies:
*   If you always strafe left, it will begin to pre-aim left.
*   If you rush aggressively, it will back up and maintain distance.
*   Higher levels introduce predictive aiming and better use of cover.

## 🚀 Installation

Ensure you have Python and Pygame installed:
```bash
pip install pygame
python main.py
```
